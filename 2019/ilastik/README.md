# Image Processing with ilastik.

## Setup & Pre-Processing 

1. Install Anaconda/miniconda. If you already have conda, skip to 2.

2. Create Conda environment: `conda create --name ilastik_fun python=3.6`

3. Install requirements: `pip install -r requirements.txt`

4. Launch Jupyternotebook: `jupyter notebook`
